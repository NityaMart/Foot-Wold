## User Guide

### For Retailers/Sellers (Mobile Only)
1. **Register**: Create account with business details
2. **Complete KYC**: Upload GST/PAN documents
3. **Browse Products**: View catalog with B2B pricing
4. **Place Orders**: Select quantities above MOQ
5. **Track Orders**: Monitor order status and delivery

### For Admins (Desktop)
1. **Admin Login**: Use credentials (admin/admin123)
2. **Dashboard**: View business metrics
3. **User Management**: Approve KYC, manage users
4. **Product Control**: Add/edit products and categories
5. **Order Oversight**: Monitor all platform orders

1. **User Selection**: Choose Retailer/Seller login
2. **Credentials**: Enter mobile/email and password
3. **OTP Verification**: Receive and enter OTP
4. **KYC Process**: Complete business verification
5. **Full Access**: Unlock all platform features
